# Quick Start

This documentation is built with [VitePress](https://vitepress.dev/).

## Run Locally

```bash
pnpm dev
```

## Build for Production

```bash
pnpm build
```

## Project Structure

```text
packages/outside-design/
├── docs/
│   ├── deliveries/            # Completed and in-progress implementation packages
│   ├── design-process/        # Design methodology and process
│   ├── development-process/   # Development workflows and local setup
│   ├── pitches/               # Proposed work before implementation
│   ├── questions/             # Open and historical design questions
│   └── architecture/          # Architecture decisions and technical notes
└── public/                    # Static assets
```
