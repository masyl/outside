## Core Runtime Libraries

### bitecs

Entity Component System (ECS) foundation for data-oriented simulation and rendering pipelines.

- **Repository**: [github.com/NateTheGreatt/bitECS](https://github.com/NateTheGreatt/bitECS)
- **License**: MIT

### Pixi.js

2D rendering engine powering game visuals and scene updates.

- **Website**: [pixijs.com](https://pixijs.com/)

- **Repository**: [github.com/pmndrs/cannon-es](https://github.com/pmndrs/cannon-es)

### Express

## Asset Generation Tools

### @napi-rs/canvas

Cairo-backed canvas rendering for Node.js, used by the bixelizer to render pixel-perfect font sprites.

- **Repository**: [github.com/nickymatts/napi-rs-canvas](https://github.com/Brooooooklyn/canvas)
- **License**: MIT

### opentype.js

TTF font parsing library used by the bixelizer to enumerate glyphs and font metrics.

- **Website**: [opentype.js.org](https://opentype.js.org/)
- **License**: MIT
