
# Standard and Custom Pointers Support

Browser standards have a list of standard names and style according to the type of user interaction is needed.

While __Outdoor__ does not support all of them yet. The default set of pointers is quite flexible.

Support for more custom pointer is also provided by the core asset packs.

## General Cursors

* ✅ default - Standard arrow cursor
* ✅ none - No cursor displayed (use carefully)
* ✅ pointer - Hand icon for clickable elements
* ❌ context-menu - Context menu available indicator

## Selection Cursors

* ✅ text - I-beam for selectable text
* vertical-text - Vertical text selection
* cell - Cell or table selection
* ✅ crosshair - Precise selection crosshair

## Drag & Drop Cursors

* ✅ move - Indicates draggable element
* ✅ grab - Open hand for grabbable items
* ✅ grabbing - Closed hand while dragging
* copy - Copy operation indicator
* alias - Shortcut/alias creation
* no-drop - Invalid drop target
* ✅ not-allowed - Action not permitted

## Resize Cursors

* col-resize - Column horizontal resize
* row-resize - Row vertical resize
* n-resize, e-resize, s-resize, w-resize - Cardinal direction 

## resizing

* ✅ ne-resize, nw-resize, se-resize, sw-resize - Diagonal resizing
* ✅ ew-resize - East-west bidirectional
* ✅ ns-resize - North-south bidirectional
* ✅ nesw-resize - Northeast-southwest bidirectional
* ✅ nwse-resize - Northwest-southeast bidirectional
* all-scroll - Omnidirectional scrolling

## Status Cursors

* ✅ wait - System busy, non-interactive
* ✅ progress - System busy but still interactive
* help - Help available indicator
* ✅ zoom-in - Zoom in available
* ✅ zoom-out - Zoom out available
