# Colors

Color palette and usage guidelines for the Outside game.

## Primary Colors

### Primary
- **Hex**: `#3B82F6`
- **Usage**: Main actions, links, highlights

### Secondary
- **Hex**: `#8B5CF6`
- **Usage**: Secondary actions, accents

## Neutral Colors

### Background
- **Light**: `#FFFFFF`
- **Dark**: `#1F2937`

### Text
- **Primary**: `#111827`
- **Secondary**: `#6B7280`

## Semantic Colors

### Success
- **Hex**: `#10B981`
- **Usage**: Success messages, positive feedback

### Warning
- **Hex**: `#F59E0B`
- **Usage**: Warnings, cautions

### Error
- **Hex**: `#EF4444`
- **Usage**: Errors, destructive actions

## Usage Guidelines

- Use primary colors for interactive elements
- Maintain sufficient contrast for accessibility
- Use semantic colors consistently across the game
