# Typography

Font families, sizes, and text styles for the Outside game.

## Font Families

### Primary Font
**Inter** - Used for body text and UI elements.

### Heading Font
**Poppins** - Used for headings and titles.

### Monospace Font
**Fira Code** - Used for code and technical content.

## Font Sizes

- **H1**: 48px / 3rem
- **H2**: 36px / 2.25rem
- **H3**: 24px / 1.5rem
- **H4**: 20px / 1.25rem
- **Body**: 16px / 1rem
- **Small**: 14px / 0.875rem
- **Tiny**: 12px / 0.75rem

## Font Weights

- **Light**: 300
- **Regular**: 400
- **Medium**: 500
- **Semibold**: 600
- **Bold**: 700

## Line Heights

- **Tight**: 1.2
- **Normal**: 1.5
- **Relaxed**: 1.75

## Usage Guidelines

- Use heading fonts for titles and section headers
- Maintain consistent hierarchy
- Ensure readable line lengths (50-75 characters)
